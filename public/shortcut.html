Paw Cities Capture

Two ways to send Instagram content to your pipeline.

How it works

1. On Instagram → Share → Copy Link
2. Open Shortcuts → tap Paw Cities
3. Done! Auto-classified and queued.

Setup (one time)

  1. Open the Shortcuts app → tap + → name it Paw Cities
  2. Add action: Get Clipboard
    Search "clipboard" → tap "Get Clipboard"
  3. Add action: Get Contents of URL
    URL: https://pawcities.com/api/ingest
    Method: POST
    Request Body: JSON
  4. Add 3 fields (tap Add New Field → Text each time):
    url → tap the value field → select the blue Clipboard variable
    source → type share_sheet
    secret → paste the key below
  5. Optional: Add action Show Notification with text "Captured!" so you get visual confirmation.

How it works

1. Copy one or more Instagram links
2. Email them to the address below
3. Auto-classified and queued within seconds.
Send links to:
ingest@pawcities.com

Tips

  • Put Instagram links anywhere in the email body — they'll all be extracted.
  • Add a city name (Paris, London, LA, etc.) in the subject to auto-tag the city.
  • Add "event" or "collab" in the subject to help classification.
  • Only emails from your authorized address are accepted.

Works from any email app on any device — iPhone, desktop, etc.
