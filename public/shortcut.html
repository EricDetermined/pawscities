Paw Cities Share

3-step setup to send Instagram content to your pipeline with two taps.

Setup (one time)

  1. Open the Shortcuts app on your iPhone
  2. Tap + then name it Paw Cities
  3. Add action: Get Contents of URL
    URL: https://pawcities.com/api/ingest
    Method: POST
    Request Body: JSON
  4. Add 3 fields (tap Add New Field → Text each time):
    url → tap value → Shortcut Input
    source → type share_sheet
    secret → paste the key below
  5. Tap the settings icon at the top → enable Show in Share Sheet

Usage

On any Instagram post or reel, tap Share → scroll to Paw Cities. Done! It auto-classifies as event, engagement, or influencer and queues it for your agents.
