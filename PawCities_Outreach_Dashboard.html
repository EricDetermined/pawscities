How to Use the Outreach Tool

  1. Research prospects for your target city (start with LA per the marketing plan)
  2. Add prospect details to a CSV file in scripts/outreach/prospects/
  3. Run a dry-run first: npx tsx scripts/outreach/send-outreach.ts --csv prospects/la.csv
  4. Review the output, then send for real: add --send flag
  5. After 5 days, run follow-ups: add --followup flag
  6. Track results in the log file at scripts/outreach/logs/outreach-log.json
